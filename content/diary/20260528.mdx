 CRUD

  🧠 背景知识：

  CRUD 是数据操作的四个基本动作的缩写：

  ┌──────┬────────┬──────┐
  │ 字母 │  英文  │ 意思 │
  ├──────┼────────┼──────┤
  │ C    │ Create │ 创建 │
  ├──────┼────────┼──────┤
  │ R    │ Read   │ 查询 │
  ├──────┼────────┼──────┤
  │ U    │ Update │ 更新 │
  ├──────┼────────┼──────┤
  │ D    │ Delete │ 删除 │
  └──────┴────────┴──────┘

  ┌───────────┬────────────────────────────────────────────────────────────────┬────────────────────────┐
  │   技术    │                             是什么                             │          类比          │
  ├───────────┼────────────────────────────────────────────────────────────────┼────────────────────────┤
  │ Egg.js    │ Node.js 服务端框架（阿里出品），处理 HTTP 请求、路由、业务逻辑 │ 相当于后端的「大脑」   │
  ├───────────┼────────────────────────────────────────────────────────────────┼────────────────────────┤
  │ Sequelize │ ORM 库，用 JavaScript 操作数据库，不用写 SQL                   │ 相当于「数据库翻译官」 │
  ├───────────┼────────────────────────────────────────────────────────────────┼────────────────────────┤
  │ socket.io │ 实时双向通信库，服务端可以主动推消息给浏览器                   │ 相当于「实时对讲机」   │
  └───────────┴────────────────────────────────────────────────────────────────┴────────────────────────┘
