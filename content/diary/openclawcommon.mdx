openclwa常用命令：

启动：openclaw gateway --port 18789

重启：openclaw gateway restart

傻瓜式配置：openclaw config

自动修复检测到的问题:openclaw doctor --fix

openclaw agents add [xxx]  添加新的agents

安装skills:clawhub install [xxx]

看完整日志：openclaw logs -f
