一些专业名词解释：

 GUI 框架                                                                                                              
  GUI = Graphical User    Interface，                                                                                                Interface，图形用户界面。就是你看到的窗口、按钮、托盘图标这些可视化的东西，区别于命令行（纯文字）


  Tauri：一个用来开发桌面应用程序的框架。类似 Electron（VS Code 用的），但更轻量。

  - Rust：Tauri 的后端逻辑用 Rust 语言编写（高性能、内存安全的系统编程语言）
  - WebView：Tauri 不自带浏览器引擎，而是调用操作系统内置的网页渲染组件（Windows 上是 Edge
  WebView2）来显示前端界面。所以前端可以用 HTML/CSS/JS 写界面，后端用 Rust 处理逻辑

  SQLite：

  一个嵌入式数据库。不需要安装数据库服务器（不像 MySQL/PostgreSQL 需要单独跑一个服务），程序直接读写一个 .db
  文件就算操作数据库了。
