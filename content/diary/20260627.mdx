## 一、调研背景

作为前端开发者，考虑转型 AI 方向。本次调研覆盖国内主流招聘网站（Boss直聘、拉勾网、Indeed）、技术社区（掘金、CSDN、知乎）以及行业报告（脉脉、InfoQ、Revealera），以下是核心发现。

---

## 二、核心结论

### 2.1 纯前端岗位确实在减少

- **全球维度**：前端岗位同比下降 **9.89%**，是软件行业中降幅最大的（Revealera 分析 1.8 亿条招聘数据，由 InfoQ 报道）
- **大厂真实动作**：
  | 公司 | 时间 | 动作 |
  |------|------|------|
  | 得物 | 2025-2026 | 前端部门解散，合入服务端做 AI 全栈 |
  | 美团 | 2025.11 | 履约前端团队并入后端 |
  | 阿里菜鸟 | 2025 | 后端全员转全栈，"前端岗将消亡" |

- **初级岗位锐减**：BOSS 直聘抽样数据显示 JD 中初级（1-2年）占比从 48% → **22%**
- **哈佛研究**：6200 万劳动者数据表明，企业采用 AI 后 6 个季度内初级开发者岗位下降 9%-10%

### 2.2 前端+AI 岗位在快速增长

- AI 相关需求占前端 JD 比例从 2024 年 15% → 2026 年 **56%**
- 大厂约 30% 前端岗位已明确要求大模型开发能力
- "AI 前端工程师"、"AI Agent 前端"等独立岗位名称已出现（阿里、联想、快手、商汤等）

### 2.3 AI 技能溢价惊人且持续扩大

| 年份 | 普通前端 | AI 前端 | 溢价 |
|------|---------|---------|------|
| 2024 | 18.6K/月 | 28.5K/月 | +53% |
| 2025 | 20.1K/月 | 35.2K/月 | +75% |
| 2026 | 21.7K/月 | 41.8K/月 | **+93%** |

两年内溢价从 53% 翻到 93%，接近翻倍。

## 三、当前 JD 核心技能要求（除 JS/CSS/HTML）

| 技能 | 要求级别 | 说明 |
|------|---------|------|
| TypeScript | **必选** | 92% 中高级岗位要求 |
| React / Vue | **必选** | 至少精通其一 |
| Node.js | **必选** | 全栈能力已成标配 |
| Next.js / Nuxt.js | **强烈推荐** | 全栈框架 |
| AI 编程工具（Cursor/Claude Code/Copilot） | **必选** | 每天在用 |
| LLM API 集成 | **强烈推荐** | 调过 OpenAI/Claude/DeepSeek |
| RAG / Agent / Function Calling / Prompt Engineering | **加分→必选** | AI 产品概念 |
| Docker / 部署 | **中高级** | BFF 部署基础 |

## 五、从培训机构 AI 课程表看学习重点

结合那张 543 个工具的 AI 课程表，前端转型只需关注：

- ✅ **大模型 API 类** — 重点学（OpenAI、Anthropic、通义等）
- ✅ **Agents 类** — 重点学（LangChain、CrewAI 等）
- ✅ **搜索引擎类** — 了解即可（RAG 场景用到）
- ⏭️ **Java 类** — 完全跳过
- ⏭️ **图像/视频生成工具** — 会用即可

## 六、个人转型计划（3 个月路线）

### 第 1 个月：AI 编程适应 + LLM API 入门
- 第 1-2 周：每天用 Cursor / Claude Code 写代码，适应 AI 编程模式
- 第 3-4 周：用 Node.js + fetch 调通 LLM API，实现流式聊天 Demo

### 第 2 个月：AI 应用项目
- RAG 知识库问答应用（PDF 上传 → 向量化 → 问答）
- 用 Next.js + Vercel AI SDK 实现

### 第 3 个月：Agent 项目
- 做一个能调用工具的简单 AI Agent
- 用 LangChain.js / LangGraph.js 实现

**当前 Node.js 水平（中下）** 足够起步，边做 AI 项目边补，不需要串行等待。
